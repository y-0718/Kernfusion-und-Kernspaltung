# Inhalte erweitern

Diese App ist so gebaut, dass du nach der Einrichtung keine Codeänderungen brauchst, um die Präsentation zu erweitern.

## Was du ohne Code ändern kannst

- Neue Slides erstellen
- Slides löschen oder archivieren
- Slides duplizieren
- Slides sortieren
- Slides veröffentlichen oder ausblenden
- Titel ändern
- Untertitel ändern
- Haupttext ändern
- Stichpunkte ändern
- Zitate ändern
- Bilder ändern
- Videos ändern
- Hintergrundbilder ändern
- Layout pro Slide auswählen
- Übergangsanimation auswählen
- Textausrichtung ändern
- Medienposition ändern
- Quellen hinzufügen
- Präsentator-Notizen bearbeiten

## Layout-Typen

Im Admin kannst du pro Slide einen Typ auswählen:

```text
Titel-Slide
Bild/Text
Vollbildbild
Video-Slide
Vergleich
Zitat/Kernaussage
Quellen
Infografik
```

## Gute Slides schreiben

Eine gute Präsentationsslide hat wenig Text.

Empfehlung:

- 1 klare Hauptaussage
- maximal 3 bis 5 Stichpunkte
- große Bilder oder Visualisierungen
- keine langen Absätze

## Medienqualität

Bilder sollten möglichst groß sein.

Empfehlung:

```text
Breite: mindestens 1600 Pixel
Format: JPG, PNG oder WebP
```

Videos:

```text
MP4 oder WebM
```

Für große Videos ist YouTube oder Vimeo oft besser als direkter Upload.

## Quellen sauber pflegen

Füge Quellen direkt bei der Slide hinzu, zu der sie gehören.

Die Quellen-Slide sammelt sie automatisch.

## Wann doch Code nötig wäre

Für normale Inhalte brauchst du keinen Code.

Code wäre nur nötig, wenn du später einen komplett neuen Layout-Typ willst, zum Beispiel:

```text
3D-Reaktor-Simulation
komplexer Quizmodus
neue Spezialanimation
eigener Drag-and-drop-Layout-Builder
```

Für die geplante Präsentation reicht die aktuelle Layout-Auswahl aus.
